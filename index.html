About

About me

I am Ceri, a software engineer in Devon, UK, although many people in the TGAA community will know me as CherryLestrade on Youtube, Twitter and Reddit. I am OBSESSED with The Great Ace Attorney Chronicles.

At the time I started this project I was a web designer and front end developer. I got to do little fun bits like this here and there but never anything on this scale. I started this project both as a way to build my skills and portfolio in the hope that I would land myself a job doing this kind of thing, and fuel my TGAA addiction while I was at it. I am pleased to report that:

  • It worked! I have a new job doing this kind of thing. I even included this project on my CV.
  • The addiction is still going strong.

The takeaway here is that TGAA made my life better, and it can make your life better too!

I would sincerely love to see any memes you create with this. Do tag me on Reddit or Twitter if you share them!

Technical details

This project was built using HTML, CSS and old-school Javascript, which was all I knew at the time I started this project. Since then I've grown in knowledge and skills, as well has having become familiar with React, Handlebars, Typescript and more. The urge to do a massive rewrite is strong. However, I've decided to leave it as it is to show that in this age of endless libraries and frameworks, you can still achieve it all in vanilla JS.

This project is hosted on Github Pages, which I'd not used before but highly recommend.

More about 'The Great Ace Attorney Chronicles'

'The Great Ace Attorney Chronicles' is a two part visual novel game by Capcom, who own the rights to all character, location and interface artwork on this page. It is honestly the best game I have ever played and I am obsessed with it despite the fact I finished it over a year ago.
